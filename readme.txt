== Description ==
 
Questo plugin aiuta la manutenzione del sito web eseguita da Hardweb.it

== Example ==

Invia informazioni sullo status del sito e comunica con i sistemi di Hardweb.it per ottimizzare il processo di manutenzione del portale.
Nessun dato personale o sensibile viene scambiato, ma solo dati tecnici relativi alle versioni installate di temi, plugin ecc...
